INTERACTION

ESC: Close the OpenGL Window
SPACE: Start/Stop animation
UP & DOWN KEYS: Change the speed of hypocycloid's animation
LEFT & RIGHT KEYS: Change the radius of the Small Circle
A & D KEYS: Change the radius of the Big Circle
W & S KEYS: Change the value of n (the number of times the Small Circle goes around the Big Circle)
Z & X KEYS: Change the scale of the hypocycloid
C & V KEYS: Rotate the hypocycloid (limit to 1 cycle)

COLLABORATORS
None

CODE SOURCES
Troy Alderson's Barebones OpenGL Program

COMPILE INSTRUCTIONS
Type 'make && ./myprogram' on terminal for the current directory.
